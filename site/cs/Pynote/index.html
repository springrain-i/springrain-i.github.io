    reduce:

    • reduce(f,s,initial)
    • f is a two-argument function
    • s is a sequence of values that can be the second argument 
      reduce(mul,[2,4,8],1)
# is equivalent to mul(mul(mul(1,2),4),8)
    • 实现reduce的代码: 
      # iteration
def reduce(f,s,initial):
    res = initial
    for i in s:
        res = f(i,res)
    return res
# recursion
def reduce(f,s,initial)
    if not s:
        return initial
    else:
        first,rest = s[0],s[1:]
        return reduce(f,rest,f(initial,first))
